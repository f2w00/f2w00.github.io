外观主题

  1. 全局主题使用moe-dark,不接受latte-dock配置,太吃内存

  2. 应用程序主题使用 kvantum-dark

    1. 安装kvantum
      1. sudo add-apt-repository ppa:papirus/papirus
        sudo apt update
        sudo apt install qt5-style-kvantum qt5-style-kvantum-themes
    2. 并使用kvantum对应用程序主题进行配置,目前使用moe-dark,主题下载方式需要到kde-store: KDE StoreKDE Store来自行下载
    3. 为了方便,可以使用ocs-url来进行安装,可以从kde-store来搜索安装

  3. plasma 视觉风格选择orchis-dark 可以优化系统托盘的图标,比较好看

  4. 图标选择fluent/tela

  5. 窗口装饰元素,我只能选择breeze,kwin窗口管理器之下,aurorae引擎的会卡

  6. 字体选择了微软雅黑,其实默认的noto sans也不错

    1. 编程字体个人喜欢source code pro

  7. terminal我选择zsh并安装oh-my-zsh,然后安装了提示插件,也可以使用yakuake,这种拉幕式终端也很好用

  8. dock/panel我选择了默认的plasma shell ,latte-dock占用内存太多,启动也有点慢

  9. 登陆屏幕选择sugar candy

  10. 桌面特效设置:

    1. magic lamp魔灯特效,很mac
    2. 透明+模糊,看起来比较高级

续航强化

安装tlp,并且手动设置自己系统的电源配置

tlp默认的设置就可以使用,个人体验下来,ubuntu开了省电模式之后续航比win11更强,平衡模式释放比win11更强

最后我的解决方案是,50%电量即开启省电模式,这样能保证续航时间

音效增强

ubuntu下默认情况下,音效比win11+nihimic情况下差不少

安装easy-effects并且在github上搜索音效并安装到easy-effects然后空间感明显增强

wayland/x11

到2024年之后应该默认都是wayland了,已经没得选了

wayland情况下多个屏幕的不同分辨率支持更好,目前没体验到不太好的地方,可能有些老软件适配不行

appimage/编译式软件建立快捷方式

  1. 右键新建一个app链接,然后打开修改路径和icon
  2. mov “from” /usr/share/applications/ 移动到系统的软件目录
  3. StartupWMClass=jetbrains-idea添加这行可以使任务栏中运行中的app和固定的图标融合
